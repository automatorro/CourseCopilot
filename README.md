# CourseCopilot

Platformă AI pentru generarea de cursuri și materiale educaționale.

## Descriere

CourseCopilot este o aplicație SaaS care permite trainerilor, profesorilor și consultanților să genereze cursuri educaționale complete pas cu pas, cu ajutorul inteligenței artificiale.

## Configurare

1.  Instalează dependențele:
    ```bash
    npm install
    ```

2.  Pornește serverul de dezvoltare:
    ```bash
    npm run dev
    ```
